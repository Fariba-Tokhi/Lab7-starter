1. Where would you fit your automated tests in your Recipe project development pipeline?

I would run automated tests in a GitHub Action whenever code is pushed because it helps catch bugs early.

2. Would you use an end to end test to check if a function is returning the correct output?

No. E2E tests check full user workflows, not small function outputs.

3. What is the difference between navigation and snapshot mode?

Navigation mode checks the page while it loads. Snapshot mode checks the current page state at one moment.
